Documentación (paso a paso)

Este documento explica el código de script.js. En resumen:

  1. Representamos números complejos como objetos {re, im} y proporcionamos operaciones aritméticas (suma, resta, producto).
  2. Para ε (Levi–Civita) generamos todas las permutaciones de columnas y evaluamos la suma sign(permutation) * Π a_{row_i, perm[i]}.
  3. Para Laplace realizamos expansión recursiva por la primera fila (o seleccionada), obteniendo menores y cofactors: |A| = Σ_j (-1)^{1+j} a_{1j} |M_{1j}|.
  4. Se muestran pasos numéricos para el ejemplo 4×4 (productos complejos desarrollados y suma final).

Si necesitas que el código genere también el desarrollo simbólico paso a paso (LaTeX), pedirlo requerirá añadir procesamiento simbólico — no incluido por petición.
